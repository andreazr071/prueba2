Student tracking System Console-based application that allows users to search for students information that allows the management of basic student information, like the registration, consultation, updating, and deletion of students, as well as the storage of information in a structured manner for later use.

This project was developed using Git and Scrum methodology.

   -Objectives 
Apply Python fundamentals Practice modular programming Work using Git branches and Pull Requests 
Simulate a real software development workflow

Features
- Register an student
- List all of them
- Edit their information
- Delete an student
-  Save their information in a csv file

  Technical Requirements 
- Error handling using try/except 
- Modular code structure 
- Interactive console menu

   Technologies Used 
- Python Git & GitHub
- Scrum methodology

   How to Run the Project
- Clone the repository
- Open the project folder Run: python src/main.py
